Autocode

Autocode can automatically add a feature to your React app. To get started, enter the information below.

Autocode will then perform the changes on your code repository, generating debugged and tested code at each step.

Note: Future versions will predict field values for target and file information based on your repository structure, saving time and allowing just the step description to complete each step. In addition, this prototype interface will be updated to support any number of steps. Currently it only allows up to 5 steps.
